Test implementation of array interface
======================================

  >>> from shapely.geometry import Point

  >>> p = Point(0.0, 0.0, 1.0)
  >>> p.array
  [0.0, 0.0, 1.0]
  >>> p.ctypes #doctest: +ELLIPSIS
  <shapely.geometry.point.c_double_Array_3 object at ...>

  Convert to Numpy array, passing through Python list

  >>> from numpy import array
  >>> a = array(p.array)
  >>> a.ndim
  1
  >>> a.size
  3
  >>> a.shape
  (3,)

  Convert to Numpy array, passing through a ctypes array

  >>> b = array(p)
  >>> b.size
  3
  >>> b.shape
  (3,)
  >>> b[0]
  0.0
  >>> b[1]
  0.0
  >>> b[2]
  1.0

  Make a point from a Numpy array

  >>> a = array([1.0, 1.0, 0.0])
  >>> p = Point(*list(a))
  >>> p.wkt
  'POINT (1.0000000000000000 1.0000000000000000)'
