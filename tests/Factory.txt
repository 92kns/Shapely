Tests of geometry factories
===========================

WKT
---

  >>> from shapely import wkt_geometry
  >>> g = wkt_geometry('POINT(0.0 0.0)')
  >>> g.geometryType()
  'Point'
  >>> g.geom_type
  'Point'
  >>> g.x
  0.0
  >>> g.y
  0.0

  Malformed WKT string should raise an error

  >>> g = wkt_geometry('POINT(0.0, 0.0)')
