Tests of geometry factories
===========================

WKT
---

  >>> from shapely import wkt_geometry
  >>> gt = wkt_geometry('POINT(0.0 0.0)')
  >>> gt.geometryType()
  'Point'
  >>> gt.geom_type
  'Point'
  >>> gt.x
  0.0
  >>> gt.y
  0.0

  Malformed WKT string should raise an error

  >>> wkt_geometry('POINT(0.0, 0.0)')
  Traceback (most recent call last):
  ...
  ReadingError: Could not create geometry because of errors while reading input.

WKB
---

  >>> from shapely import wkb_geometry
  >>> POINTA = "010100000000000000000000000000000000000000".decode("hex")
  >>> gb = wkb_geometry(POINTA)
  >>> gb.geom_type
  'Point'
  >>> gb.x
  0.0
  >>> gb.y
  0.0
